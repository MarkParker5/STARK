# Core Concepts

The pieces every command is built from, regardless of how simple or complex it gets:

- **[Patterns](patterns.md)**: the syntax for matching what a user says and pulling parameters out of it.
- **[Command Response](command-response.md)**: what a command returns: text, voice, status, and how to chain into follow-up commands.
- **[Commands Context](commands-context.md)**: nested menus, follow-ups, and stateful conversations.
- **[Dependency Injection](dependency-injection.md)**: getting response handlers, language info, and your own dependencies into a command function.

Read them in order if you're new, each builds a bit on the last.
